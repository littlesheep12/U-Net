# U-NET
## ENVIRONMENT
pycharm+python3.6+pytorch1.3.1  
## HOW TO RUN:
Enter the dataset.py and correct the path of the datasets
example:
```
python main.py --action train&test --arch UNet --epoch 21 --batch_size 21 
```
